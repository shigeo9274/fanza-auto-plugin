== Description ==


DMM WEB APIを利用してFANZAアフィリエイト記事の大量自動投稿を行うプラグインです。


== Installation ==

1.プラグインzipファイルを用意してください。
2.Wordpress管理画面→新規プラグインを選択して下さい。
3.ファイルアップロードから「fanza-auto.zip」をアップロードして下さい。
4.適用をクリックして下さい。
